Script to determine the input parameters for generating the REF_APP_FluxConversion reference file¶

This script can be used to generate the plots presented in the following document: https://redmine.isdc.unige.ch/projects/cheops/wiki/Flux_Conversion

In addition to generating the plots, the script performs a fit to the empirical ratio (observed flux / predited flux) vs effective temperature. The parameters from this fit are used as input to the runFluxConversion executable in the CHEOPSim SOC software package, used to generate the REF_APP_FluxConversion reference file.

An initial predicted flux (before empirical correction for observed/preducted vs TEff) is determined by generating an uncorrected version of REF_APP_FluxConversion, by executing runFluxConversion with the following non-default settings in runFluxConversion.xml:

CH_TU2020-01-29T00-00-00_REF_APP_FluxConversion_V0104_nofit.fits:

  • fitIntercept = 1.
  • fitSlope = 0.

The above configuration models stellar spectra using PHOENIX SEDs for TEff<7200K, and black body spectra for higher TEffs. For comparison, a version using black body spectra for all effective temperatures is generated by running with the following non-default settings:

CH_TU2020-01-29T00-00-00_REF_APP_FluxConversion_V0104_nofit_blackbody.fits

  • SEDFilename = null
  • blackBodyOffset = 0.
  • fitIntercept = 1.
  • fitSlope = 0.

It is the comparison of value of (CHEOPS mag - Gaia mag) at TEff=7200K from these two cases which provides the default value for blackBodyOffset in runFluxConversion.xml

Obtain (CHEOPS mag - Gaia mag) for each star by interpolating between the points in CH_TU2020-01-29T00-00-00_REF_APP_FluxConversion_V0104_nofit.fits to the effective temperature of the star.

In [34]:
CminusG_Andrea = []
for star in range(len(TEff_Andrea)):
    CminusG = 0
    for i in range(len(TEff)):
        if TEff[i] > TEff_Andrea[star]:
            mu = (TEff_Andrea[star]-TEff[i-1])/(TEff[i]-TEff[i-1])
            CminusG = CmagMinusGmag[i-1]*(1.-mu)+CmagMinusGmag[i]*mu
            break
    CminusG_Andrea.append(CminusG)

Calculate the predicted flux for each star as F0_X 10^(-0.4CHEOPSmag) where F0_X is defined in https://redmine.isdc.unige.ch/projects/cheops/wiki/Flux_Conversion and has been read as a header keyword of CH_TU2020-01-29T00-00-00_REF_APP_FluxConversion_V0104_nofit.fits
