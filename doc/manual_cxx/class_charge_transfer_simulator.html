Detailed Description

This module simulates the charge transfer efficiency (CTE)

Author
David Futyan UGE

The effect of CTE is assumed to be cumulative and linear, such that the number of electrons in a given pixel after charge transfer is equal to the initial number of electrons multiplied by the product of the CTE and the number of row/column shifts
