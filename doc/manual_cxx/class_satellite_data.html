Detailed Description

Container for information relating to the satellite conditions at a given time (temperature, roll angle, jitter, stray light, particle flux)

Author
David Futyan UGE

A vector of SatelliteData (each vector element corresponds to a different time step) can be accessed from the Data class via Data::getSatelliteData()
