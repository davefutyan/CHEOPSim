Detailed Description

This module generates a static dark frame (shot noise is added later by PhotonNoiseGenerator) with mean value depending on CCD temperature, and generates hot, warm and telegraphic pixels.

Author
David Futyan UGE

The expected dark current depends on the CCD temperature and takes into account the total time during which pixels can accumulate dark current, including readout time.
