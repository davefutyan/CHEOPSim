Detailed Description

Container class for time dependent star data.

Author
David Futyan UGE

A vector of pointers to StarData (one element for each time step) is provided in the Star class.

Each instance of StarData (corresponding to a given time step) provides for that time step the multiplicative factors to be applied to the flux of a Star (relative to Star::meanFlux() or Star::meanFlux(double wavelength)
