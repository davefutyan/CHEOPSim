Detailed Description

This module projects the positions of stars within the field of view onto the focal plane of the detector using the fits_world_to_pix World Coordinate System routine.

Author
David Futyan UGE

The gnomonic (tangent plane) projection and plate scale of 1 pixel per arcsecond are used. The projection algorithm uses the roll angle as a function of time defined in OrbitSimulator.
