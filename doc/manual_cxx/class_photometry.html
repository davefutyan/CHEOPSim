Detailed Description

Class to perform photometric extraction from an Image.

Author
David Futyan UGE

Parameters for the photometric extraction are passed to the constructor. The setFlatField method can optionally be used to define a flat field correction during photometric extraction. Photometric extraction is performed using the extractFlux method. Methods are also provided to evaluate the PSF barycentre and the median background.
